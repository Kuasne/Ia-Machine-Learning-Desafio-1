Desafio 1 - Machine Learning com Python | Aprendizado Supervisionado

Este vídeo apresenta o desenvolvimento do Desafio 1 da disciplina de Inteligência Artificial / Machine Learning, realizado como componente avaliativo da NPC1.

🎯 **Objetivo:**
Desenvolver um projeto completo de aprendizado supervisionado, aplicando todas as etapas do processo de Machine Learning: escolha e análise exploratória do dataset, pré-processamento dos dados, treinamento de modelo, avaliação de desempenho e validação com conjunto de teste.

📋 **Etapas realizadas:**
- Análise e compreensão do conjunto de dados
- Pré-processamento e preparação dos dados
- Seleção e treinamento de algoritmo de aprendizado supervisionado
- Avaliação da acurácia e validação do modelo

🛠️ **Tecnologias e ferramentas utilizadas:**
- Python
- Google Colab
- Bibliotecas: pandas, numpy e scikit-learn
